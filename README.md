# Ultrasonic-obstacle-avoidance-Bluetooth-car
This 51 MCU course design completes basic functions like LED, nixie tube, LCD1602, matrix keyboard and ADC0832 acquisition. A Bluetooth smart car with obstacle avoidance and PWM speed control is built with TB6612 and 25GA-370 motors, meeting all requirements.

一： Project Overview
This course design project is based on the STC89C51 chip. It implements functions such as breathing light, PWM control, nixie tube display and LCD display. It is extended to a Bluetooth ultrasonic obstacle-avoidance car.
二： Experimental Equipment
STC89C51, STM32F103C8T6, LCD screen, nixie tube, Bluetooth module BT08B
三： Experimental Environment
Keil5
四： Obstacle Avoidance Algorithm
Ultrasonic ranging
